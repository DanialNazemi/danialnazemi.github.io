## 🛠️ Tech Stack

This portfolio is powered by:

- **[Gulp 5](https://gulpjs.com/)** - Task automation
- **[Sass/SCSS](https://sass-lang.com/)** - Advanced CSS preprocessing
- **[Babel](https://babeljs.io/)** - Modern JavaScript transpilation
- **[Browserify](http://browserify.org/)** - Module bundling
- **[Autoprefixer](https://autoprefixer.github.io/)** - Automatic vendor prefixes
- **[ImageMin](https://github.com/imagemin/imagemin)** - Image optimization
- **Google Analytics** - Performance tracking
